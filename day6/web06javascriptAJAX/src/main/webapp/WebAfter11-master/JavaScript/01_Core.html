1. HTML : 웹페이지를 제작하기 위한 도구
         2. CSS : HTML 표현(꾸밈)
         3. 자바스크립트 :
            1) HTML 요소(tag, element) 제어한다
            2) 동적화(DOM, Document of Model) 용도 사용
            3) 다양한 라이브러를 사용
            4) 웹 애플리케이션을 만듬, 서버 구성 가능

         4. 자바스크립트 구성
            1) 자바스크립트 CORE : 기본 문법 구조 (데이터 타입, 조건문, 반복문, 배열, 함수 등)
            2) 자바스크립트 BOM (Browser Object Model) : 브라우저와 관련된 Window, Navigator, Laction, History, Document 등
            3) 자바스크립트  DOM (Document Object Model) : Element 제어, Style 제어, Attribute 제어, Event 제어
